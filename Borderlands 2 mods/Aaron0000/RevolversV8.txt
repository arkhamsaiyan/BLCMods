########
By Aaron0000

Features;
Changes the Jakobs pistol barrel to behave like the Revolvers of Borderlands 1. Also treats the Maggie, Greed, Rex, and Unforgiven in the same way (May want to disable these for balance).

High damage.
High accuracy.
+100% critical hit damage.
Lowered fire rate.
Lowered reload speed.
Recoil is larger.
Consumes 3 ammo per shot (Maliwan consumes 4 per shot.)
All revolvers have an 18 ammo clip, excluding Vladof (36), Maliwan (24), and Bandit (45).

NOTES:

If using the Luck Cannon (either version), there is a fix for it at the bottom.
If using S&S Forever, these is a fix for it at the bottom.
If using the Defiler (my version), there is a fix for it at the bottom.
If using Atlas Reborn, there is a fix for it at the bottom. There is also an additional fix for the Aries.

If using both Atlas and S&S, there is a combined fix at the bottom.
